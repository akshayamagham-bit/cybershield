🔐 Password Security

Learn how to protect your accounts from hackers

📌 What is Password Security?

Password security means creating and managing strong passwords to protect your accounts from unauthorized access.

⚠️ Why is it Important?

  • Protects personal data
  • Prevents hacking
  • Keeps accounts safe

✅ Best Practices

  • Use 8–12 characters minimum
  • Mix uppercase, lowercase, numbers, symbols
  • Use different passwords for each account
  • Enable 2-factor authentication (2FA)

❌ Common Mistakes

  • Using "123456" or "password"
  • Using your name or DOB
  • Reusing same password everywhere

💡 Pro Tip

Use a password manager to generate and store secure passwords.
